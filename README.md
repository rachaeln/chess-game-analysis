# Chess Game Analysis

One of the world's most popular games, chess has entertained millions since the inception of its current form in the 15th century. [Lichess](https://lichess.org) is a free and open-source internet chess server where one can play chess, mull over puzzles, and compete in tournaments. Using [this dataset](https://www.kaggle.com/datasnaek/chess) from Kaggle, over 20,000 chess games from Lichess were analyzed with R.
